# GPS Telematics — Web: Functional Document

> **Product**: Asset Rental Platform — Vehicle Variant
> **Domain**: GPS Telematics

---

## 1. Purpose & Scope

GPS Telematics and the Live Map are entirely Desk/Fleet Manager features. **There are no customer-facing web requirements.** Estimated current mileage for GPS vehicles is covered in the Mileage & Fuel domain.

---

## 2. Requirements

> None. GPS data is not shown to customers on any web page.
